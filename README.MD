Dag 1 diskuterade vi idéer för vår webbapplikation, efter lite fram och tillbaka kom vi överens om att vi skulle bygga en BMI kalkylator. Därefter byggdes ett alldeles för stort projekt med AI-verktyg, så vi bestämde att skala ner projektet och ta bort det AI genererade och börja på ett helt nytt repo. Med en del miss i kommunikationen där det inte va helt klart för alla vad som skulle göras tog vi ett möte och klarade ut komplikationerna, med hjälp av discord och kommunikation mellan gruppmedlemmarna så flöt projektet på bra tills slutgiltig produkt. Sprintplaneringen höll som den var designad, vi la till lite ytterligare funktionalitet. En inloggning/registrering sida, och ett formulär för att hantera uppgifter.
Sista dagen, alla samlades och vi testade funktionaliteten och sökte efter buggar, la till ytterligare features som vi ansåg var nödvändiga som en, logout knapp och navigationer till respektive sidor med mera. Sedan skrevs det dokumentation.

Produktbeskrivning:
Inloggning och registrering med Localstorage .
BMI-kalkylator som säknar ut bmi och ger möjligheten att spara sina resultat,
Även ett ”dummy” formulär där man kan ställa ”frågor”.
Inehåller testning för bmi.ts, bmi-dom.ts, bmi-utils.ts

Behöver Köras frö att applikationen ska fungera

npm i
npm run build
